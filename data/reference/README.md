# Reference Data



This folder contains small reference files committed for reproducibility.



## Files



### smart_sample_20260523_073153.csv

The priority image sample generated by `pipeline\smart_sample.py` on 22 May 2026.

4,842 products selected using three-tier sampling strategy.

See `docs/ADR.md` ADR-010 for sampling rationale.



### vision_results_20260523_210131.csv

Azure Vision OCR + GPT-4.1-nano claim extraction results.

4,842 products processed. 2,008 with front-of-pack claims detected.

Generated by `pipeline\vision_extract.py` on 23 May 2026.

Cost: ~7.74 CHF (Azure Vision S1 + Azure OpenAI).



## Reproducing from scratch



1. Download OFF bulk export: `python pipeline\download_export.py`
2. Run full pipeline: `python pipeline\bulk_load.py`
3. Generate smart sample: `python pipeline\smart_sample.py`
4. Run vision extraction: `python pipeline\vision_extract.py`
