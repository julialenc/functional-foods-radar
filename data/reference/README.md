# Reference Data



This folder contains small reference files committed for reproducibility.



## Files



### smart\_sample\_20260523\_073153.csv

The priority image sample generated by `pipeline\\smart\_sample.py` on 22 May 2026.

4,842 products selected using three-tier sampling strategy.

See `docs/ADR.md` ADR-010 for sampling rationale.



### vision\_results\_20260523\_210131.csv

Azure Vision OCR + GPT-4.1-nano claim extraction results.

4,842 products processed. 2,008 with front-of-pack claims detected.

Generated by `pipeline\\vision\_extract.py` on 23 May 2026.

Cost: \~7.74 CHF (Azure Vision S1 + Azure OpenAI).



## Reproducing from scratch



1. Download OFF bulk export: `python pipeline\\download\_export.py`
2. Run full pipeline: `python pipeline\\bulk\_load.py`
3. Generate smart sample: `python pipeline\\smart\_sample.py`
4. Run vision extraction: `python pipeline\\vision\_extract.py`
